# Tomcat



* 왜 tomcat 서버가 필요한가?
  * 브라우저는 html 문서만 읽을 수 있다. Html 형식에 java 코드가 섞인 jsp 파일의 java 코드는 읽을 수 없다. **브라우저는 java를 스스로 컴파일 하지 못한다. 브라우저가 java 파일을 컴파일 하기 위해서 tomcat(WAS) 서버가 필요하다.**

>
