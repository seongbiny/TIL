# 시맨틱 마크업

HTML 태그들 중 "의미가 있는 태그"

### `시맨틱태그가 중요한 이유`

1. SEO ( Search engine optimization )

검색엔진이 시맨틱태그를 중요한 키워드로 간주하기 때문에 검색엔진 최적화에 유리하다.

2. 웹 접근성

음성으로 읽어주는 '스크린리더'를 이용하거나, 키보드만을 이용해서 웹사이트를 이용하는 경우 적절한 시맨틱 태그로 잘 만들어진 웹사이트라면 문제없이 동작할 수 있다.

3. 유지보수성

다른 개발자들이 코드를 유지보수하기 더 편해진다.

| tag     | summary                      |
| ------- | ---------------------------- |
| header  | 헤더                         |
| nav     | 네비게이션 바                |
| aside   | 사이드에 위치하는 공간       |
| section | 여러 중심 내용을 감싸는 공간 |
| article | 글자가 많이 들어가는 부분    |
| footer  | 푸터                         |
